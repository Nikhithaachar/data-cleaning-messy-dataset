Data Cleaning Project (Messy Dataset)

Description

This project focuses on cleaning a messy dataset containing employee information such as name, age, email, salary, and join date.

Data Issues

- Missing values in age and salary
- Duplicate records
- Incorrect data types (age stored as text)
- Invalid email formats
- Inconsistent date formats

Tasks Performed

- Loaded dataset using Pandas
- Removed duplicate records
- Converted age column from text to numeric
- Filled missing age values using average
- Filled missing salary values using median
- Cleaned and validated email column
- Standardized date format
- Exported cleaned dataset

Tools Used

- Python
- Pandas

Output

Cleaned dataset saved as: "cleaned_data.csv"# data-cleaning-messy-dataset
